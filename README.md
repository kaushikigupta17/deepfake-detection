## What we built
A deepfake detection model that classifies face images as REAL or FAKE.

## Approach
- Used EfficientNetB0 with Transfer Learning
- Trained on 140,000 face images (70k real + 70k fake)
- Achieved 89.5% validation accuracy

## Dataset
140k Real and Fake Faces - Kaggle

## Results
- Validation Accuracy: 89.5%
- Test predictions: predictions.csv
